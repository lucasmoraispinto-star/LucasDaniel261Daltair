# Grand Theft Auto VI - Fan Page Project 🎮✨

Bem-vindo ao repositório da Fan Page oficial conceitual do **Grand Theft Auto VI (GTA 6)**. Este projeto é uma aplicação web interativa desenvolvida para reunir informações, novidades e elementos visuais inspirados no trailer e na atmosfera de Vice City do próximo grande lançamento da Rockstar Games.

---

## 🎯 Objetivo do Projeto
O principal objetivo deste projeto é criar uma plataforma moderna, imersiva e visualmente impactante que centralize os principais detalhes revelados sobre o GTA 6. Além do foco na experiência do usuário (UX/UI) inspirada nas luzes de neon e na cultura de Miami/Vice City, o projeto serve como um portfólio prático para consolidar conhecimentos em desenvolvimento web front-end.

## 🚀 Tecnologias Utilizadas
O projeto foi construído utilizando as tecnologias fundamentais da web, sem a necessidade de frameworks externos, priorizando a performance e a fidelidade visual:

* **HTML5:** Estruturação semântica de todo o conteúdo da página (seções, cabeçalhos, listas e formulários).
* **CSS3:** Estilização avançada com o uso de variáveis (Custom Properties), gradientes lineares, efeitos de sombra (`box-shadow` e `text-shadow`), transições suaves e responsividade total (Mobile First).
* **JavaScript (ES6+):** Inserção de interatividade dinâmica, como um contador regressivo para o lançamento, filtragem de notícias, manipulação do DOM e animações disparadas por eventos de clique e rolagem.

## 🌟 Funcionalidades
* **Design Temático (Cyber Neon):** Identidade visual baseada na paleta de cores oficial do trailer (tons de roxo, rosa neon e azul ciano).
* **Contador Regressivo (Countdown):** Um cronômetro interativo em JavaScript que exibe os dias, horas, minutos e segundos restantes para a previsão de lançamento do jogo.
* **Seção de Protagonistas Alternável:** Seção interativa onde o usuário pode alternar entre as informações e histórias dos dois protagonistas: *Lucia* e *Jason*.
* **Galeria de Novidades Dinâmica:** Filtros interativos controlados via JavaScript para categorizar as novidades por "Plataformas", "História" e "Jogabilidade".
* **Formulário de Newsletter:** Área para os fãs se inscreverem e receberem atualizações, com validação de campos via JavaScript.
* **Totalmente Responsivo:** Layout adaptável para smartphones, tablets, notebooks e desktops.
